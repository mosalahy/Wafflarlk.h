# Field Force Manager

Production-ready Arabic-first field sales management platform with real-time location tracking.

## Features

- Real-time location tracking
- Role-based access control
- Arabic RTL interface
- PWA support
- Admin dashboard with live map
- Visit logging with attachments
- Attendance tracking
- Audit logging

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000/login`
